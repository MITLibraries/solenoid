Instructions for generating CSV files

Windows

These instructions have been tested on Office 2016. If you have a different version, your actual workflow may vary.

  • Open your file in Excel
  • Click the File menu
  • Choose Save As
  • Browse to the folder you want to save in and double click to select it
  • You get a standard Windows pop up that gives options for file name, file type, and even navigation to choose a different folder.
  • Type in the file name (it can be anything that works for you; the app does not care)
  • In the Save as type: dropdown list, select CSV (Comma delimited) (*.csv)
  • Click Save
  • Excel hates text files, so you get a warning: Some features in your workbook might be lost if you save it as CSV; do you want to keep using that format?
  • Click Yes
  • When you try to close or save the file, even if you haven’t made any edits, you’ll get this warning again. Click Yes to proceed.

Mac

  • Open your file in Numbers. Excel will not work if your file has any sort of special characters (math symbols, diacritics, etc.)
  • Click the File menu
  • Choose Export to... and then CSV...
  • Save with whatever file name you want.
  • You don't need to use the advanced options; the defaults work fine.
